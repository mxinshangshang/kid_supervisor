# 快速启动指南

## 环境要求

- 树莓派5 (推荐8GB版本)
- Camera Module 3 (或其他兼容摄像头)
- Python 3.9+

## 1. 检查摄像头

首先确认摄像头工作正常：

```bash
libcamera-hello
```

如果能看到画面，说明摄像头正常。

## 2. 检查依赖

树莓派系统通常预装了必要的库，确认一下：

```bash
python3 -c "import cv2; print('OpenCV OK')"
python3 -c "import picamera2; print('Picamera2 OK')"
```

如果缺少依赖，安装：

```bash
sudo apt update
sudo apt install -y python3-opencv python3-picamera2 python3-numpy
```

## 3. 运行程序

### 方式一：直接运行（推荐）

```bash
cd /home/mxin/.openclaw/workspace/kid_supervisor_v3
python3 main.py
```

### 方式二：使用虚拟环境

```bash
cd /home/mxin/.openclaw/workspace/kid_supervisor_v3
source venv/bin/activate
python3 main.py
```

## 4. 测试人脸检测（无头模式，无GUI）

如果想在SSH远程测试，可以运行：

```bash
python3 test_face_detection.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| `q` / `ESC` | 退出程序 |

## 功能说明

- **人脸检测**：使用 OpenCV DNN 深度学习模型，NEON 加速
- **学习计时**：检测到人脸3帧后开始计时，人脸消失3帧后停止
- **实时显示**：显示人脸框、FPS、学习时长、检测引擎

## 故障排除

### 摄像头找不到

```bash
# 检查摄像头
libcamera-hello
# 检查摄像头设备
ls -l /dev/video*
```

### 无法显示窗口

如果是SSH连接，需要开启X11转发：

```bash
ssh -X pi@树莓派IP
```

或者先测试无头模式：`python3 test_face_detection.py`

### 性能问题

- 如果 FPS 太低，可以在 `main.py` 中调整 `FRAME_SIZE` 为更小的值（如 480x360）
